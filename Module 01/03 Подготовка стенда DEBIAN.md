### **Задача 1: Настройка Виртуальной машины Debian**

Настройка server

Debian предназначен для мониторинга

   
1. На панели инструментов нажмите кнопку "Запустить".

2. Войдите в систему под учетной записью указанной сверху.

3. Запустите терминул.

4.Получите права супер пользователя.
```
student@debian:~$ su -
```
```
< Введите пароль пользователя student >
```

5. Изменить имя компьютера
```
hostnamectl set-hostname  debian.corp.local
```
6. Откройте в текстовом редакторе файл /etc/hosts.

7. Замените вторую строку на
```
127.0.1.1 debian debian.corp.local
```

8. Перезапустить сетевые адаптеры
```
systemctl restart networking
```
9. Проверка IP адресации
```
ip a
ip r
```
10. Проверка связи
```
ping ya.ru
```
